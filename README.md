# 为霜

#### 介绍
- 对于领创平板管理的研究方案

#### 软件架构
1. 用于建立对TB3—850F（科大讯飞定制版）的第三方管控
2. 基于系统内**framework.jar**所提供的接口
3. 功能（已实现）
    - 安装科大讯飞未认证应用
    - 隐藏应用
    - 打开USB调试
    - 解锁TF卡使用权限
    - 允许设置锁屏密码
    - 允许打开文件管理
    - 允许打开WIFI高级设置
4. 增强功能
    - 天气查询（每天一百次刷新机会）
    - 增强系统日历，自动导入中国农历及节假日等内容

#### 安装教程

1. 请于**领创平板管理**初始化前输入初始密码安装
2. 请于日历应用中输入 **www.weishuang.com** 唤起应用


#### 使用说明

1. 本应用采用内部账号使用策略。
