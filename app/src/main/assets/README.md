# Pad Manager

#### 软件简介
1. 用于代替领创平板管理控制平板
2. 基于接口
3. 功能（已实现）
    - 安装科大讯飞未认证应用
    - 隐藏应用
    - 打开USB调试
    - 解锁TF卡使用权限
    - 允许设置锁屏密码
    - 允许打开文件管理
    - 允许打开WIFI高级设置
    - 允许关闭网络防火墙
    - 蓝牙
